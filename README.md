# Icon Downloader 图标下载器

一个简单易用的网站图标批量下载工具，支持多种下载服务，可以自动获取网站的高清图标。

## 功能特点

- 🎯 **多种输入方式**
  - 直接输入网址（如：www.google.com）
  - 输入品牌名称（如：微信、支付宝）
  - 支持任意中英文名称，系统会自动搜索对应网站

- 🔍 **多个下载源**
  - Google Favicon（需要特殊网络环境）
  - DuckDuckGo（国内可用）
  - Yandex（国内可用）
  - ico.kucat.cn
  - 直接从目标网站下载（自动尝试多种路径）

- 🎨 **图标质量保证**
  - 支持多种尺寸：64x64、128x128、256x256、512x512
  - 自动保持原始尺寸（如果大于目标尺寸）
  - 保留图标透明度
  - 使用高质量的重采样算法

- 💡 **智能功能**
  - 自动域名解析
  - 批量下载支持
  - 自动重试机制
  - 详细的下载状态提示

## 安装说明

### 方法一：直接使用

1. 下载 `dist` 目录中的 `IconDownloader.exe`
2. 双击运行即可使用，无需安装

### 方法二：从源码运行

1. 确保已安装 Python 3.8 或更高版本
2. 安装依赖：
```bash
pip install -r requirements.txt
```
3. 运行程序：
```bash
python icon_downloader_gui.py
```

## 使用方法

1. 启动程序后，在输入框中输入要下载的网址或名称，每行一个
2. 选择需要的图标尺寸（64~512像素）
3. 选择要使用的下载服务（可多选）
4. 点击"开始下载"按钮
5. 等待下载完成，可以在"下载结果"区域查看详细信息
6. 点击"打开下载文件夹"查看下载的图标

## 注意事项

- Google Favicon 服务需要特殊网络环境
- 建议国内用户优先使用 DuckDuckGo 和 Yandex 服务
- 下载失败时会自动重试，最多重试3次
- 如果原始图标尺寸大于选择的尺寸，将保持原始尺寸
- 所有图标都会被转换为PNG格式，并保持透明度
